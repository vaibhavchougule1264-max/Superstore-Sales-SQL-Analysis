# 📌 Project Overview

This project analyzes the Superstore Sales dataset using MySQL and Python. The objective is to perform data cleaning, exploratory data analysis (EDA), business analysis, and data visualization to generate meaningful business insights.

---

## 🛠️ Technologies Used

- MySQL 8.0
- MySQL Workbench
- Python 3.x
- Pandas
- SQLAlchemy
- PyMySQL
- Matplotlib
- Seaborn
- Jupyter Notebook
- Git & GitHub

---

## 📂 Dataset

**Dataset Name:** Superstore Sales Dataset

The dataset contains approximately **10,000 sales records** with the following information:

- Order Details
- Customer Details
- Product Details
- Sales
- Profit
- Quantity
- Discount
- Region
- Category
- Sub-Category
